Basic Proportionality Theorem (BPT / Thales Theorem):
If a line is drawn parallel to one side of a triangle, it divides the other two sides proportionally.
If DE‚ą•BC in ‚Ė≥ABC, then AD/DB = AE/EC.
Converse: If AD/DB = AE/EC, then DE‚ą•BC.

Q1. In ‚Ė≥ABC, DE‚ą•BC. Find EC or DB using BPT:
(i) AD=1.5cm, DB=3cm, AE=1cm. Find EC.
(ii) AD=4cm, AE=8cm, EC=3cm. Find AB.
Answer:
(i) AD/DB=AE/EC ‚Üí 1.5/3=1/EC ‚Üí EC=2cm. ‚úÖ
(ii) AD/DB=AE/EC ‚Üí 4/DB=8/3 ‚Üí DB=3/2=1.5cm. AB=AD+DB=5.5cm. ‚úÖ
